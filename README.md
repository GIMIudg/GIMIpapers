# GIMIpapers

GIMIpapers is the official repository of the *Grupo de Ingeniería Metabólica In Silico (GIMI)*. This repository compiles code and resources from research projects developed within the group.

Our work spans multiple domains, including metabolic engineering, precision medicine, and social-oriented STEM initiatives. The repository contains implementations in Python, MATLAB, and SQL.

Each project includes a brief description and, when available, a link to the corresponding publication.
